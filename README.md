## Tech Stack
* Typescript
* React
* Redux
* Redux Toolkit
* Emotion (object styles)
* WebSocket

## Launch
1. Start the server
```shell
# Navigate to the 'server' folder
yarn start
```
2. Start the app
```shell
# Navigate to the 'client' folder
yarn app:start
```
